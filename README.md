# python-course-notebooks-pl
Interaktywne notebooki do nauki Pythona — teoria, zadania i auto-sprawdzanie odpowiedzi.

## Jak korzystać z tej dokumentacji?

| Potrzebujesz... | Idź do... |
| :--- | :--- |
| **Nauczyć się Pythona** (lekcje i zadania) | [Folder z modułami](moduly/) |
| **Dowiedzieć się, jak dodać/edytować zadania** | [Przewodnik dla twórców](docs/how-to/jak-dodac-nowy-modul.md) |
| **Sprawdzić techniczne detale API** | [Dokumentacja techniczna](docs/reference/checker-api.md) |
| **Zrozumieć, jak to działa pod spodem** | [Wyjaśnienie architektury](docs/explanation/architektura.md) |

## Struktura repozytorium

```text
.
├── checker.py              # Framework do sprawdzania zadań
├── cwiczenia/              # Pliki z definicjami zadań (.py)
├── moduly/                 # Interaktywne notebooki (.ipynb)
├── docs/                   # Dokumentacja systemu
└── _szablony/              # Gotowe wzorce do tworzenia nowych modułów
```

## Szybki start dla ucznia

Jeśli jesteś tu po to, żeby przejść kurs:
1. Wejdź do folderu [moduly/](moduly/).
2. Otwórz wybrany notebook.
3. Kliknij przycisk **Open in Colab**.
4. Szczegółową instrukcję znajdziesz tutaj: [Jak uruchomić notebook w Colab](docs/how-to/jak-uruchomic-notebook-w-colab.md).

Powodzenia! 🐍
